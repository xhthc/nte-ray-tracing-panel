# 快速使用

## 推荐方式

下载 Release 里的单文件 exe：

```text
NTERayTracingPanel-v版本号.exe
```

双击运行后会自动打开：

```text
http://127.0.0.1:22642
```

如果没有自动打开，手动复制上面的地址到浏览器。

## 页面步骤

1. 点击“选择”，选择《异环》安装根目录，或选择 `Client\WindowsNoEditor\HT\Binaries\Win64`。
2. 点击“检测”，确认识别到 `HTGame.exe`。
3. 点击“下载/准备 OptiScaler”。
4. 关闭游戏和启动器，或勾选“安装/恢复前自动关闭异环和启动器”。
5. 点击“备份并安装光追解锁”。
6. 启动游戏，进入画质设置检查“光线追踪 / 全景光追”。

## 退出工具

这个工具分成两层：

- 前端 WebUI：浏览器里的页面。
- 后端服务：`NTERayTracingPanel.exe` 或 `python app.py`。

关闭浏览器标签页不会退出后端服务，端口 `22642` 仍会被占用。需要退出时点击页面底部“退出工具”。如果页面已经关了，可以重新打开 `http://127.0.0.1:22642` 后再点“退出工具”，或者在任务管理器里结束 `NTERayTracingPanel.exe`。
