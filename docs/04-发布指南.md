## 构建

```bat
build_exe.bat
```

输出目录：

```text
dist\NTERayTracingPanel\
```

如果要做单文件版，可以手动运行：

```powershell
python -m PyInstaller --noconsole --onefile --name NTERayTracingPanel --add-data "web;web" app.py
```

## 发布说明应写清楚

- 本工具只监听 `127.0.0.1:22642`。
- 关闭浏览器不等于退出 exe。
- 默认不修改显卡注册表。
- 每次安装都会备份。
- 需要退出时点“退出工具”或结束 `NTERayTracingPanel.exe`。
