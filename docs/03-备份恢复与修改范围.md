# 备份恢复与修改范围

## 修改范围

默认 DXGI 模式只管理游戏 `Win64` 目录内的文件：

```text
winmm.dll
OptiScaler.ini
OptiScaler.log
OptiScaler\
dlsstweaks.ini
dlsstweaks.log
```

其中 `dlsstweaks.*` 只是为了兼容你之前做过 DLSS 面板实验时的备份恢复，不会作为光追解锁核心写入。

## 不会默认修改

```text
HKLM\SYSTEM\CurrentControlSet\Enum\PCI\...\DeviceDesc
HKLM\SYSTEM\CurrentControlSet\Enum\PCI\...\FriendlyName
```

也不会写 NVIDIA App 全局设置。

## 备份位置

每次安装都会创建：

```text
<Win64>\_nte_rt_backups\<时间戳>
```

目录内包含：

```text
manifest.json
files\
```

`manifest.json` 记录安装前哪些文件存在、备份位置、文件哈希、安装模式和操作步骤。

## 恢复逻辑

恢复不是简单删除当前文件，而是按 manifest 执行：

- 安装前存在的文件：从备份复制回去。
- 安装前不存在但由工具创建的文件：恢复时删除。
- 安装前存在的目录：从备份目录恢复。
- 安装前不存在的目录：恢复时移除。

这保证恢复目标是“回到那次安装前的状态”。
