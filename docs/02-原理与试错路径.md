# 原理与试错路径

## 为什么能解锁光追

《异环》当前测试版本会按 GPU 型号白名单决定是否显示光线追踪选项。当前机器的 `RTX 5060 Laptop GPU` 通过 OptiScaler DXGI/Streamline spoof 成 `RTX 4090` 后，游戏内光线追踪选项已经可以打开。

## 默认可用路径

默认安装路径是：

```text
Win64\winmm.dll
Win64\OptiScaler.ini
Win64\OptiScaler\
```

`winmm.dll` 是本地代理入口。它只在游戏加载该本地 DLL 时生效。

## 不推荐的路径

- 不再使用 ProcMon 检测游戏行为，因为它会加载 `PROCMON` minifilter，容易被游戏保护识别为黑客工具。
- 不默认改系统显卡注册表，因为这是整机级影响。
- 不默认启用 Full 模式，因为它会增加 Registry/User32/FakeNVAPI spoof 层。

## 如果 DXGI 模式失败

1. 确认 `OptiScaler.log` 是否生成。
2. 确认 `OptiScaler.ini` 中 `TargetProcessName=HTGame.exe`。
3. 确认游戏和启动器不是在安装后被热修复覆盖文件。
4. 恢复备份后重装。
5. 最后再尝试 Full 实验模式。
