# Security

## Local-Only Service

The panel listens on `127.0.0.1:22642` by default. It is intended for local use only.

## Reporting Issues

When reporting a problem, include:

- Windows version.
- GPU model and driver version.
- Whether `fltmc filters` shows `PROCMON`.
- The selected game `Win64` path.
- `OptiScaler.log` tail if it exists.
- The backup ID from `_nte_rt_backups`.

Do not include account tokens, personal documents, or unrelated logs.
